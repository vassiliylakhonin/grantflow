# Kazakhstan NGO Pilot External Proof Summary

## Pilot Shape

Bounded pilot across three donor tracks:
- `UN agencies`
- `EU`
- `USAID`

Pilot scope:
- `3` live cases
- measured baseline captured before launch
- workflow operated through live drafting, review, and final-package cycles

## What The Pilot Was Testing

The pilot was designed to test whether GrantFlow improves proposal operations on the metrics that matter most to institutional proposal teams:
- time to first reviewable draft
- total time to final package
- review-loop count
- duplicate or conflicting review comments
- late-stage MEL / LogFrame rework

## What Was Confirmed

### Strongest proof

`UN agencies` and `EU` produced the clearest proof of value.

These paths showed:
- materially faster movement to a reviewable first draft
- fewer full review loops
- lower duplicate/conflicting comment burden
- fewer late-stage MEL rewrites
- cleaner movement to final package

### Hardest path

`USAID` also improved against baseline, but remained the heaviest coordination path.

The pilot still showed value on `USAID`, but not at the same operating level as `UN` and `EU`.

## Outcome Read

Pilot verdict:
- `continue_with_conditions`

Why:
- the pilot clearly validated the operating model
- two donor paths produced strong evidence
- the remaining weaker path still improved materially, but should be described with caution rather than treated as fully solved

## Practical Product Read

This pilot supports the claim that GrantFlow is useful as:
- a proposal operations control layer
- a structured draft + review workflow
- a MEL / LogFrame coordination aid
- an export-ready review package generator

It does **not** support the claim that every donor workflow is already equally mature.

The honest read is:
- strong proof on `UN/EU`
- positive but conditional proof on `USAID`

## What This Is Good For Externally

This pilot is strong enough to support:
- buyer conversations with NGOs and implementing organizations
- bounded pilot discussions for similar proposal teams
- a commercial narrative focused on workflow control, review traceability, and faster reviewable drafts

Recommended wording:
- GrantFlow showed strongest pilot evidence on `UN` and `EU` proposal workflows, while the `USAID` path improved materially but remained more coordination-heavy and should still be positioned with conditions.
