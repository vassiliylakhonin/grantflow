# Kazakhstan NGO Donor Corpus Plan

This plan captures the donor-specific corpus agreed after kickoff for the first pilot wave.

## Goal

Prepare enough donor and case evidence for the first three pilot cases so grounded drafting is credible from day one.

## In-Scope Donors

1. `un_agencies`
2. `eu`
3. `usaid`

## UN Agencies Corpus

Include:
- latest relevant UN templates for concept notes and proposals
- budget guidance if applicable
- evaluation criteria or scoring guidance where available
- MEL guidance for results framework, indicators, targets, and disaggregation
- safeguarding / GESI / do-no-harm requirements
- standard annex checklist
- de-identified successful prior proposals if permitted

Corpus owner:
- Zhanna + Ruslan

Validation owner:
- Timur

Readiness target:
- `2026-03-13`

## EU Corpus

Include:
- call package
- guidelines for applicants
- annex templates
- evaluation grids
- logical framework matrix guidance
- budget eligibility and compliance notes
- cross-cutting requirements such as gender, climate, and rights-based approach
- `1-2` anonymized historical submitted packages

Corpus owner:
- Ruslan

Validation owner:
- Asel + Zhanna

Readiness target:
- `2026-03-13`

## USAID Corpus

Include:
- NOFO / RFP instructions
- technical narrative and annex instructions
- ADS / CLA / MEL references
- indicator and results-framework expectations
- compliance packet essentials
- internal lessons from previous submissions
- standard technical approach and implementation structure

Corpus owner:
- Timur + Asel

Validation owner:
- Aidana delegate

Readiness target:
- `2026-03-14`

## Corpus Acceptance Rule

Do not mark a donor track ready until:
- core donor guidance is loaded
- at least one example of working structure or prior package is available if permitted
- MEL / LogFrame guidance is present
- the responsible validator signs off that the pack is usable for a live run
