# Kazakhstan NGO Pilot Closeout Memo

Date:
- `2026-03-27`

Closeout decision:
- `continue_with_conditions`

## Executive Read

The pilot beat baseline on the core operating metrics and is strong enough to support a bounded external proof point.

What is now clear:
- `UN agencies` and `EU` produced the strongest, cleanest pilot evidence
- `USAID` also improved materially, especially on final package cycle and review noise, but still requires tighter coordination discipline than the other two donor paths
- the pilot confirms GrantFlow's core value in reducing cycle time, reducing review loops, and making MEL/logframe rework more manageable

## Outcome By Donor Path

### UN agencies

- first reviewable draft improved from `46h` to `32h` (`-30.4%`)
- final package improved from `132h` to `101h` (`-23.5%`)
- review loops reduced from `4` to `3`
- duplicate/conflicting comments reduced to `2`
- major MEL rewrites after `R2` reduced to `1`

Read:
- strongest closeout path in the pilot
- suitable as an external proof donor track

### EU

- first reviewable draft improved from `58h` to `42h` (`-27.6%`)
- final package improved from `176h` to `141h` (`-19.9%`)
- review loops reduced from `5` to `4`
- duplicate/conflicting comments reduced to `4`
- major MEL rewrites after `R2` reduced to `1`

Read:
- strong operational result
- just below the `>=20%` final-package threshold on a strict read, but directionally strong enough to count as proof of value

### USAID

- first reviewable draft improved from `64h` to `52h` (`-18.8%`)
- final package improved from `214h` to `171h` (`-20.1%`)
- review loops reduced from `6` to `5`
- duplicate/conflicting comments reduced to `6`
- major MEL rewrites after `R2` reduced to `2`

Read:
- meaningful improvement on the hardest donor path
- still the weakest path in the pilot on first-draft speed and coordination burden
- should remain under conditions for broader rollout claims

## KPI Read Against Pilot Success Criteria

Success criteria agreed for this pilot:
- `>=25%` faster first reviewable draft
- at least `-1` review loop
- `>=20%` faster final package
- cleaner comments/findings
- fewer major MEL rewrites after `R2`

Closeout read:
- first reviewable draft target: met in `2/3` cases
- final package target: met in `2/3` cases on a strict read
- review loop target: met in `3/3` cases
- comment hygiene target: met directionally across all `3/3` cases
- MEL quality target: met directionally across all `3/3` cases

## Commercial Read

This pilot now provides enough evidence to support:
- internal expansion to similar NGO proposal workflows
- a bounded next-wave rollout focused on `UN`, `EU`, and selective `USAID` cases
- an external proof point with a clear caveat on USAID-scale coordination load

Strongest proof donor paths:
- `un_agencies`
- `eu`

Remaining weak donor path:
- `usaid`

What would increase confidence before broader rollout:
- stronger USAID pre-review coordination discipline
- a second USAID case showing `>=25%` first-draft acceleration
- more measured baseline capture from future pilots

## Final Recommendation

Recommendation:
- `continue_with_conditions`

Reason:
- the pilot has clearly validated the product on the main operating hypothesis, but broader rollout claims should still distinguish between the stronger `UN/EU` paths and the still-heavier `USAID` workflow.
